# 日記草案（AI日記）生成（v1）

ユーザーとの会話と構造化メモをもとに、その日の **AI日記** を Markdown で出力する。

会話ログの直前に、エントリ日と生成時点の「今日」の差分に応じた指示が付く。エントリ日が今日でないときは「今日のまとめ」を「この日のまとめ」に置き換え、本文は過去形。「明日」は当時の翌日や、いま振り返った次の一歩に言い換える。

必須の見出しテンプレ（空なら省略可）:

- `## 今日のまとめ（AI）`
- `### 出来事`
- `### 感情`
- `### 気づき`
- `### 明日（次の一歩）`
- `### 予定との関係（任意）`

本文は丁寧すぎず自然な日本語。事実と推測を混同しない。
